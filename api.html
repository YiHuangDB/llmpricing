LLM Pricing API Documentation

Access real-time LLM pricing data programmatically through our static JSON API.

View Pricing Table Download JSON

Quick Start

GET https://[your-username].github.io/llmpricing/data/llm-pricing.json

JavaScript Example

// Fetch LLM pricing data
fetch('https://[your-username].github.io/llmpricing/data/llm-pricing.json')
  .then(response => response.json())
  .then(data => {
    // Get all OpenAI models
    const openai = data.providers.find(p => p.name === 'OpenAI');
    console.log('OpenAI models:', openai.models);
    
    // Find cheapest model
    const allModels = data.providers.flatMap(p => 
      p.models.map(m => ({...m, provider: p.name}))
    );
    const cheapest = allModels.sort((a, b) => 
      a.pricing.input - b.pricing.input
    )[0];
    console.log('Cheapest model:', cheapest);
  });

Python Example

import requests
import json

# Fetch LLM pricing data
response = requests.get('https://[your-username].github.io/llmpricing/data/llm-pricing.json')
data = response.json()

# Get all Anthropic models
anthropic = next((p for p in data['providers'] if p['name'] == 'Anthropic'), None)
if anthropic:
    for model in anthropic['models']:
        print(f"{model['name']}: ${model['pricing']['input']}/1M tokens")

# Find models with vision capabilities
vision_models = []
for provider in data['providers']:
    for model in provider['models']:
        if 'Vision' in model.get('features', []):
            vision_models.append({
                'provider': provider['name'],
                'model': model['name'],
                'price': model['pricing']['input']
            })

Data Structure

Root Object

Field Type Description
lastUpdated string (ISO 8601) Timestamp of last data update
providers array List of LLM providers

Provider Object

Field Type Description
name string Provider name (e.g., "OpenAI", "Anthropic")
website string Provider's pricing page URL
models array List of available models

Model Object

Field Type Description
name string Display name of the model
modelId string API identifier for the model
contextWindow number Maximum context length in tokens
pricing object Pricing information
features array List of model capabilities
category string Model category (flagship, efficient, etc.)

Pricing Object

Field Type Description
input number Cost per 1M input tokens (USD)
output number Cost per 1M output tokens (USD)
unit string Pricing unit (usually "per 1M tokens")
note string (optional) Additional pricing information

Common Use Cases

Find Cheapest Model

const cheapest = data.providers
  .flatMap(p => p.models)
  .sort((a, b) => 
    a.pricing.input - b.pricing.input
  )[0];

Filter by Context Window

const longContext = data.providers
  .flatMap(p => p.models)
  .filter(m => 
    m.contextWindow >= 100000
  );

Get Vision Models

const visionModels = data.providers
  .flatMap(p => p.models)
  .filter(m => 
    m.features.includes('Vision')
  );

Calculate Token Cost

function calculateCost(model, inputTokens, outputTokens) {
  const inputCost = (inputTokens / 1000000) * model.pricing.input;
  const outputCost = (outputTokens / 1000000) * model.pricing.output;
  return inputCost + outputCost;
}

Model Categories

Models are categorized to help you find the right balance of performance and cost:

flagship Most capable models with highest performance
efficient Cost-effective models for high-volume use
balanced Good balance of capability and cost
reasoning Specialized for complex reasoning tasks
open-source Open source models available through cloud providers

Update Frequency

The pricing data is automatically updated daily at 2:00 AM UTC through GitHub Actions.

Updates are fetched from official provider websites and APIs where available.

Rate Limits & Usage

This is a static JSON file hosted on GitHub Pages with no rate limits.

Feel free to cache the data locally and check the lastUpdated field for freshness.

Recommended Caching Strategy

const CACHE_DURATION = 3600000; // 1 hour in milliseconds

async function getPricingData() {
  const cached = localStorage.getItem('llm-pricing');
  
  if (cached) {
    const data = JSON.parse(cached);
    const age = Date.now() - new Date(data.cachedAt).getTime();
    
    if (age < CACHE_DURATION) {
      return data;
    }
  }
  
  const response = await fetch('https://[your-username].github.io/llmpricing/data/llm-pricing.json');
  const data = await response.json();
  
  localStorage.setItem('llm-pricing', JSON.stringify({
    ...data,
    cachedAt: new Date().toISOString()
  }));
  
  return data;
}
